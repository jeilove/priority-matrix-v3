
<==**메인 화면**==>

상단에는 모든 할일, AI 리포트, 설정 화면으로 가는 메뉴가 있고 
바로 아래에 1사분면, 2사분면, 3사분면, 4사분면의 화면으로 가는 메뉴가 있어

동그라미 모양의 영역 중앙에 할일 입력 버튼 배치하고 입력하면 중앙에 표시
- 할일 내용과 예상소요시간을 입력받는다.

동그라미에 연속으로 입력할 시, 최근 입력 내용만 보이고 이전 것은 저장, 좌우 버튼으로 이전 내용 표시, 
더블클릭하면 전체 입력 내용 볼 수 있는 전체 화면으로 이동, 나가기 버튼 누르면, 메인 화면(동그라미 있는)으로 이동

동그라미 모양 주변을 4개 동등한 영역으로 분할
- 1사분면 :  "긴급+중요" 표시. 다른 2~4사분면도 동일하게 적용 #즉시 처리할 일(마감 있는 일, 중요일)
- 2사분면 :  "중요+긴급하지 않음" 표시 #계획 수립(자기계발, 장기 계획)
- 3사분면 :  "긴급+중요하지 않음" 표시 #위임(단순 반복 업무)
- 4사분면 :  "긴급하지 않음+중요하지 않음" 표시 #삭제(단순 유튜브시청, 웹서핑, 불필요업무)

각 1~4분면을 더블클릭 하면 각 사분면에 입력된 내용을 표시하는 화면으로 전환, 나가기 선택하면 다시 메인 원과 사분면이 보이는 곳으로 이동

중앙 원에서 입력된 내용은 드래그앤 드랍이 가능한 형태로 제시하여 1~4사분면 어느 곳이나 드래그 가능하게 구현

<==**모든 할일 화면>

전체 할일을 관리할 수 있는 종합 화면
상단에 할일 추가 버튼(할일과 우선순위 입력 가능한 화면 제시 )
카드 형태로 모든 할일 들을 표시(최근 갱신 순으로 제시)
- 아직 진행하지 않은 것은 우선순위 4가지 분류로 표시
- 그밖의 것은 상태에 따라 진행중, 완료, 보류중으로 표시될 수 있음 

하단에 최근 활동 로그
- 할일 완료, 보류중에 대한 것들만 할일 제목과 상태정보(완료, 보류중) 제시

==1사분면 화면

상태에 따라 진행중, 완료, 보류중으로 표시



* AI 기능
AI 자동 분류 : 사용자가 중앙 원형에 "내일까지 세금 계산서 발행하기"라고 입력하고, AI 자동 분류 메뉴 클릭하면 AI가 자연어를 분석하여 **1사분면(긴급+중요)**으로 시각적인 추천(하이라이트 효과 등)
업무 패턴 및 생산성 리포트 : 주간/월간 리포트를 통해 "이번 주는 3사분면(긴급하지만 중요하지 않음)에 시간을 너무 많이 썼네요. 단순 업무는 위임하거나 줄여보세요"와 같은 맞춤형 피드백을 제공
작업 잘게 쪼개기 기능: 예) 2사분면에 "영어 공부하기"처럼 모호한 목표를 넣으면, AI가 "하루 단어 10개 외우기", "10분 팟캐스트 듣기" 등 실천 가능한 세부 태스크로 쪼개어 제안   

* 모바일 앱으로 개발시 : 음성 메모 기능

설정 메뉴
1. 테마 및 디스플레이: 다크모드/라이트모드 설정, 1~4사분면 색상 커스터마이징(예: 긴급함은 빨간색, 계획은 파란색 등).
2. 알림 설정: 1사분면 마감일 임박 푸시 알림, 아침/저녁 일간 브리핑 알림 시간 설정.
3. 데이터베이스 연동: 구글 시트 연동(DB 내보내기/가져오기).
4. AI 설정: AI 자동 분류 기능 켜기/끄기
